# SnapKarte（スナップカルテ） サービス概要書

## 1. サービスコンセプト
**「シャッターを切るだけ、営業終了。」**

SnapKarteは、美容室・サロンの現場業務を極限までシンプルにする、LINE自動連携型のCRM（顧客管理）プラットフォームです。施術後の写真撮影という日常業務をトリガーに、顧客へのお礼メッセージ送信・カルテ保存・次回予約の促進をすべて自動化します。

## 2. 解決する課題 (Pain Points)
美容師は「技術」以外の事務作業に多くの時間を奪われています。

*   **連絡の負担**
    営業終了後、疲れた体で一人一人にお礼LINEを手動送信している負担を解消します。
*   **写真共有の手間**
    AirDropやQRコードでの写真共有はお客様の端末や設定に依存し、時間がかかります。SnapKarteなら撮影するだけで完了します。
*   **カルテ管理の限界**
    紙のカルテやスマホのカメラロールでは、過去のスタイル履歴を即座に引き出せません。SnapKarteはクラウド上で整理・保存します。

## 3. 主要機能 (Key Features)

### ① 1-Tap LINE連携（特許出願級UX）
美容師がアプリで施術写真を撮影・保存するだけで、連携済みのLINE公式アカウントから、お客様へ自動的にお礼メッセージと写真が送信されます。
*   **自動化**: 送信作業ゼロ。文章作成の手間もゼロ。
*   **即時性**: お客様が退店してすぐに「感動」を届けられます。

### ② スマート・スタイルカルテ
顧客ごとの施術履歴（写真、日付、メニュー、メモ）が時系列で自動整理されます。
*   **瞬時検索**: 名前や来店日から、過去のスタイルを3秒で呼び出し。カウンセリングの質が向上します。
*   **タグ管理**: 「カット」「カラー」「ショート」などのタグで直感的に整理できます。

### ③ QRチェックイン（会員証機能）
お客様のスマホに表示された会員証QRコードを読み取るだけで、受付とカルテ呼び出しが完了します。
*   **検索不要**: 名前を聞き返すことも、検索窓に入力する必要もありません。

### ④ サロン経営支援（Standardプラン）
*   **スタッフ管理**: 複数のスタイリストを管理し、誰が担当したかを記録。
*   **リピート分析**: 最終来店日からの経過日数を可視化し、失客防止のアラートなどの施策につなげられます。

## 4. ターゲット市場
*   **個人経営・フリーランス美容師**: 事務作業を減らし、接客と技術に集中したい層。
*   **小規模ヘアサロン（1〜5席）**: 高価なPOSシステムを入れるほどではないが、DXを進めたい店舗。

## 5. ビジネスモデル
*   **Freeプラン (¥0)**: 月間送信数10件まで。導入のハードルを極限まで下げ、利用体験を広めます。
*   **Soloプラン (¥2,980/月)**: 送信数無制限。個人事業主向けのスタンダードプラン。
*   **Standardプラン (価格未定)**: 複数スタッフ管理、高度な分析機能などを提供予定。

## 6. 技術スタック
*   **Frontend**: Next.js (App Router), React, TailwindCSS
*   **Backend**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
*   **Infrastructure**: Vercel (Edge Network)
*   **Payment**: Stripe Connect (Subscription)
*   **Integration**: LINE Messaging API, LIFF

## 7. 今後のロードマップ
*   **AIカウンセリング**: 過去の履歴から「次はこんなスタイルどうですか？」と提案文を自動生成。
*   **EC連携**: 施術で使ったシャンプーなどを、届いたLINEからそのまま購入可能に。
