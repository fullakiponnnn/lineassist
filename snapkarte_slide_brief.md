## 1. 概要

| 項目 | 内容 |
|---|---|
| 資料タイトル | SnapKarte — 個人スタイリストのためのデジタルカルテ |
| 対象 | 個人美容師・フリーランス美容師・面貸しスタイリスト |
| 目的 | サービス理解 → 無料登録への誘導 |

## 2. デザインシステム

### カラーパレット

| 用途 | カラー | HEX |
|---|---|---|
| プライマリ（深緑） | ██ | `#134231` |
| プライマリ薄 | ██ | `#2d5a47` |
| サーフェス（背景） | ██ | `#fbf9f5` |
| テキスト（主） | ██ | `#1b1c1a` |
| テキスト（副） | ██ | `#414944` |
| アクセント（ゴールド） | ██ | `#6a5e33` |
| LINE グリーン | ██ | `#06C755` |
| CTA / 強調 | ██ | `#c17a2f`（テラコッタ） |
| 白 | ██ | `#ffffff` |

### タイポグラフィ

| 用途 | フォント | サイズ目安 |
|---|---|---|
| スライドタイトル | **Noto Serif JP** Bold | 36〜44pt |
| サブタイトル | Noto Sans JP Medium | 20〜24pt |
| 本文 | Noto Sans JP Regular | 16〜18pt |
| キャプション / 注釈 | Noto Sans JP Light | 12〜14pt |
| 英語ラベル | Inter / Outfit Semi-Bold | 12〜16pt |

### デザイントーン

- **エディトリアル × ミニマル**: 余白を十分にとり、高級感を演出
- **写真中心**: 各スライドに1枚はビジュアルを配置
- **角丸**: カード要素は `16〜24px` の角丸
- **影**: ドロップシャドウは控えめ（`0 8px 30px rgba(0,0,0,0.04)`）
- **装飾線**: 使用しない。区切りは余白で表現
